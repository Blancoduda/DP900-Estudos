# Questões de Revisão

## Questão 1

Qual formato é mais indicado para armazenar dados analíticos de forma otimizada?

- A) CSV
- B) JSON
- C) Parquet
- D) XML

✅ **Resposta:** C

---

## Questão 2

Qual serviço do Azure é um banco de dados NoSQL totalmente gerenciado?

- A) Azure SQL Database
- B) Azure Cosmos DB
- C) Azure Blob Storage
- D) Azure Synapse Analytics

✅ **Resposta:** B

---

## Questão 3

Qual propriedade ACID garante que uma transação será concluída totalmente ou cancelada completamente?

- A) Consistência
- B) Isolamento
- C) Atomicidade
- D) Durabilidade

✅ **Resposta:** C

---

## Questão 4

Qual tecnologia é utilizada para criar medidas no Power BI?

- A) SQL
- B) Python
- C) DAX
- D) M

✅ **Resposta:** C

---

## Questão 5

Qual serviço da Microsoft reúne Data Engineering, Data Warehouse, Data Science e Power BI em um único ambiente?

- A) Azure SQL Database
- B) Azure Synapse Analytics
- C) Microsoft Fabric
- D) Azure Cosmos DB

✅ **Resposta:** C
