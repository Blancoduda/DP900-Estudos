# 📊 08 - Visualização e Análise com Power BI

## 🎯 Objetivo

Compreender como o Microsoft Power BI permite transformar dados em informações úteis por meio de modelos semânticos, relatórios, dashboards e visualizações interativas para apoiar a tomada de decisão.

---

# O que é o Microsoft Power BI?

O **Microsoft Power BI** é uma plataforma de Business Intelligence (BI) utilizada para conectar, transformar, modelar e visualizar dados provenientes de diversas fontes.

Seu principal objetivo é transformar dados em informações que auxiliam empresas na tomada de decisão.

Com o Power BI é possível:

- Conectar diferentes fontes de dados;
- Criar modelos de dados;
- Desenvolver relatórios interativos;
- Criar dashboards;
- Compartilhar análises com outros usuários.

---

# Componentes do Power BI

O ecossistema Power BI é composto por diferentes componentes.

## Power BI Desktop

Aplicação instalada no computador utilizada para:

- Importar dados;
- Transformar dados;
- Criar modelos;
- Desenvolver relatórios.

---

## Power BI Service

Serviço online utilizado para:

- Publicar relatórios;
- Compartilhar dashboards;
- Gerenciar workspaces;
- Agendar atualizações.

---

## Power BI Mobile

Aplicativo para Android e iOS.

Permite acompanhar dashboards e relatórios diretamente no celular.

---

# Conectando Dados

O Power BI pode se conectar a diversas fontes de dados.

Exemplos:

- Azure SQL Database
- Azure Synapse Analytics
- Microsoft Fabric
- Azure Blob Storage
- Azure Data Lake Storage
- Excel
- CSV
- SharePoint
- APIs
- Bancos SQL

---

# Transformação de Dados

Antes da análise, normalmente os dados precisam ser preparados.

No Power BI isso é feito utilizando o **Power Query**.

Entre as principais transformações estão:

- Remover colunas;
- Renomear campos;
- Alterar tipos de dados;
- Mesclar tabelas;
- Filtrar registros;
- Criar colunas calculadas.

---

# Modelo Semântico

Após a preparação dos dados, é criado um **Modelo Semântico**.

Esse modelo organiza os dados para facilitar consultas e análises.

Ele normalmente contém:

- Tabelas fato;
- Tabelas dimensão;
- Relacionamentos;
- Medidas;
- Hierarquias.

---

# Relacionamentos

Os relacionamentos conectam tabelas diferentes.

Exemplo:

Clientes

↓

Pedidos

↓

Produtos

Esses relacionamentos permitem criar análises mais completas.

---

# Medidas

As medidas são cálculos realizados durante a análise.

São normalmente criadas utilizando a linguagem **DAX (Data Analysis Expressions)**.

Exemplos:

- Soma de vendas;
- Média de faturamento;
- Quantidade de pedidos;
- Percentuais.

---

# Relatórios

Um relatório é composto por uma ou mais páginas contendo visualizações.

Pode incluir:

- Gráficos;
- Tabelas;
- Cartões;
- Mapas;
- Segmentações;
- Indicadores.

---

# Dashboards

Um dashboard reúne diversos visuais em uma única tela.

Seu objetivo é apresentar indicadores importantes de forma resumida.

É muito utilizado para acompanhamento executivo.

---

# Visualizações

O Power BI oferece diversos tipos de visualizações.

Entre elas:

- Gráfico de barras;
- Gráfico de linhas;
- Pizza;
- Área;
- Dispersão;
- Mapas;
- Cartões;
- Matrizes;
- KPIs.

---

# Compartilhamento

Após publicar um relatório no Power BI Service é possível:

- Compartilhar com outros usuários;
- Criar workspaces;
- Definir permissões;
- Agendar atualização dos dados.

---

# Integração com Azure

O Power BI integra-se naturalmente com diversos serviços do Azure.

Exemplos:

- Microsoft Fabric
- Azure Synapse Analytics
- Azure SQL Database
- Azure Data Lake Storage
- Azure Blob Storage
- Azure Databricks

---

# Comparação

| Recurso | Função |
|----------|--------|
| Power BI Desktop | Criar relatórios |
| Power BI Service | Publicar e compartilhar |
| Power BI Mobile | Acessar relatórios |
| Power Query | Transformar dados |
| Modelo Semântico | Organizar os dados |
| DAX | Criar medidas e cálculos |
| Dashboard | Monitorar indicadores |
| Relatório | Explorar informações |

---

# Resumo

- Power BI é uma plataforma de Business Intelligence.
- Power Query prepara os dados.
- O Modelo Semântico organiza os dados para análise.
- DAX é utilizado para criar medidas.
- Relatórios permitem explorar dados.
- Dashboards apresentam indicadores resumidos.
- O Power BI integra-se aos principais serviços do Azure.
